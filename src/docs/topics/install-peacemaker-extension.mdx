Follow these steps to install the Peacemaker browser extension:

### Step 1: Open the Browser Extension Store

- For Chrome: Open the [Chrome Web Store](https://chrome.google.com/webstore).
- For Firefox: Open the [Firefox Add-ons](https://addons.mozilla.org).

### Step 2: Search for "Peacemaker"

In the search bar, type "Peacemaker" and press Enter.

### Step 3: Select the Peacemaker Extension

Click on the Peacemaker extension from the search results.

### Step 4: Add to Browser

- For Chrome: Click the "Add to Chrome" button.
- For Firefox: Click the "Add to Firefox" button.

### Step 5: Confirm Installation

A dialog will appear asking for confirmation. Click "Add Extension" to confirm.

### Step 6: Verify Installation

Once installed, you should see the Peacemaker icon in your browser's toolbar.

Congratulations! You have successfully installed the Peacemaker extension.
