# Documentação do Projeto

Bem-vindo à documentação do nosso projeto! Este é um exemplo de como você pode usar MDX para criar documentação interativa.

## O que é MDX?

MDX é uma extensão do Markdown que permite que você use componentes React diretamente em seus arquivos de documentação.

## Exemplo de Código

```jsx
import React from 'react';

function MeuComponente() {
  return <div>Olá, mundo!</div>;
}
```

## Recursos

- [Documentação oficial do MDX](https://mdxjs.com)
- [Guia de Markdown](https://www.markdownguide.org)

## Como Usar

1. Crie arquivos `.mdx` na pasta `src/docs`
2. Importe componentes React quando necessário
3. Use sintaxe Markdown para formatação
4. Adicione exemplos de código quando relevante
